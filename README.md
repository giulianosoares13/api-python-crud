# api-python-crud

**Status:** Em desenvolvimento

## Objetivo
API REST simples em Python para operações CRUD (Create, Read, Update, Delete).  
Projeto criado para aprendizagem de backend, conceitos de rotas, persistência com banco de dados e testes básicos.

## Tecnologias propostas
- Python 3.x
- FastAPI (ou Flask)
- SQLAlchemy (ou outro ORM) / ou uso direto de psycopg2 para PostgreSQL
- PostgreSQL (ou SQLite para desenvolvimento)
- Uvicorn (para execução local)
- pytest (testes básicos)
